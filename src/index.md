Dive Into Python 3
==================

Dive Into Python 3 covers Python 3 and its differences from Python 2.
Compared to [Dive Into Python](http://diveintopython.org/), it’s about
20% revised and 80% new material. The book is now complete, but
[feedback is always welcome](about.html).
