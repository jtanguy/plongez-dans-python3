Things to Read
--------------

Unfortunately, I can not write cover every facet of Python 3 in this
book. Fortunately, there are many wonderful, freely available tutorials
available elsewhere.

Decorators:

-   [Function
    Decorators](http://programmingbits.pythonblogs.com/27_programmingbits/archive/50_function_decorators.html)
    by Ariel Ortiz
-   [More on Function
    Decorators](http://programmingbits.pythonblogs.com/27_programmingbits/archive/51_more_on_function_decorators.html)
    by Ariel Ortiz
-   [Charming Python: Decorators make magic
    easy](http://www.ibm.com/developerworks/linux/library/l-cpdecor.html)
    by David Mertz
-   [Function
    Definitions](http://docs.python.org/reference/compound_stmts.html#function)
    in the official Python documentation

Properties:

-   [The Python `property`
    builtin](http://adam.gomaa.us/blog/2008/aug/11/the-python-property-builtin/)
    by Adam Gomaa
-   [Getters/Setters/Fuxors](http://tomayko.com/writings/getters-setters-fuxors)
    by Ryan Tomayko
-   [`property()`
    function](http://docs.python.org/library/functions.html#property) in
    the official Python documentation

Descriptors:

-   [How-To Guide For
    Descriptors](http://users.rcn.com/python/download/Descriptor.htm) by
    Raymond Hettinger
-   [Charming Python: Python elegance and warts, Part
    2](http://www.ibm.com/developerworks/linux/library/l-python-elegance-2.html)
    by David Mertz

-   [`threading` — Manage concurrent

-   [`multiprocessing`

-   [`multiprocessing` — Manage processes like

-   [Python threads and the Global Interpreter

Metaclasses:

-   [Metaclass programming in
    Python](http://www.ibm.com/developerworks/linux/library/l-pymeta.html)
    by David Mertz and Michele Simionato

    2](http://www.ibm.com/developerworks/linux/library/l-pymeta2/) by
    David Mertz and Michele Simionato

    3](http://www.ibm.com/developerworks/linux/library/l-pymeta3.html)

In addition, Doug Hellman’s [Python Module of the
Week](http://www.doughellmann.com/PyMOTW/contents.html) is a fantastic
guide to many of the modules in the Python standard library.

Where To Look For Python 3-Compatible Code {#code}
------------------------------------------

As Python 3 is relatively new, there is a dearth of compatible
libraries. Here are some of the places to look for code that works with
Python 3.

-   [Python Package Index: list of Python 3
    packages](http://pypi.python.org/pypi?:action=browse&c=533&show=all)
-   [Python Cookbook: list of Python 3
    recipes](http://code.activestate.com/recipes/langs/python/tags/meta:min_python_3/)
-   [Google Project Hosting: list of projects tagged
    “python3”](http://code.google.com/hosting/search?q=label:python3)
-   [SourceForge: list of projects matching “Python
    3”](http://sourceforge.net/search/?words=%22python+3%22)
-   [GitHub: list of projects matching
    “python3”](http://github.com/search?type=Repositories&language=python&q=python3)
    (also, [list of projects matching “python
    3”](http://github.com/search?type=Repositories&language=python&q=python+3))
-   [BitBucket: list of projects matching
    “python3”](http://bitbucket.org/repo/all/?name=python3) (and [those
    matching “python 3”](http://bitbucket.org/repo/all/?name=python+3))
